# 🧠 GenAI Support Copilot (RAG + Spec Generator)

A production-ready **GenAI application** that combines:

- 🔍 **RAG-based Question Answering (Task 1)**
- 🧩 **AI-powered Software Spec Generator (Task 2)**

Built using **FastAPI, LangChain, FAISS, Hugging Face LLM (Mistral 7B), and React**.

---

## 🚀 Live Demo

🔗 https://dinesh-05-rag-support-copilot.hf.space

---

## 📌 Features

### 🔹 Task 1: RAG Support Assistant
- ✅ Context-based question answering  
- ✅ FAISS similarity search (Top-K retrieval)  
- ✅ Hallucination control using threshold  
- ✅ LLM-enhanced answer generation  

---

### 🔹 Task 2: AI Spec Generator
- ✅ Converts raw requirements into:
  - Features
  - User Stories
  - API Endpoints
  - Database Schema  
- ✅ Structured output generation  
- ✅ Prompt-engineered pipeline  
- ✅ Powered by Mistral 7B LLM  

---

### 🔐 Authentication
- ✅ Signup / Login / Logout  
- ✅ Session-based authentication (FastAPI middleware)
